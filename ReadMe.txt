To start this project ,

1.You need to have python3.6 or Higher versions and pip3 (python3.6 version)

2.You need MongoDB 3.8 or higher 


3. run the following command -

	>>> pip3 install -r requirements.txt


4.CKYC2>CKYC>setings.py


DATABASE_LOCAL='ABC_DB'  // name of your database (example 'ABC_DB')

CENTRAL_SERVER='********'  // ip_address of ckyc-central-server

BANK_NAME='XYZ_Bank'  // name of your bank (example 'SBI','UBI')

CKYC_AUTH_TOKEN='***********************'  // give the token that is generated by central_server to you.


DATABASES = {
    'default': {
        ..........,
        'NAME': 'DATABASE_LOCAL',
        ...........,
    }
}









3.Then goto project folder and run the server using following cmd
	>>> python3 manage.py runserver 8000

4.If its first time you are running it will for ask for enter Service name , 
	Eg.
	>>>  enter service name :cKYC_central

5.now do migrations using following :

	>>> python3 manage.py makemigrations 
	>>> python3 manage.py migrate
	
6.now create superuser using cmd ,
	>>> python3 manage.py createsuperuser 

this user is authorised to login and create policy

7.Once policy confirmed and you will get a option to download xml file
